Design notes:
--- Major tool are: VScode ； HTML Language；PHP Language；Javascript Language；Wampserver；PHPAdmin；MYSQL; Echarts...
--- Use VScode software to edit HTML language to create pages
--- Html文件在VScode里面可以直接按：Ctrl+1，打开浏览器预览 （需要下载安装插件“open in browser”
--- PHP文件为服务器文件，不能直接打开，在浏览器中输入地址打开：如 http://localhost/Testchartshtml.php
--- Right click on page, and Use "View page source" to see the codes
--- Use CSS code to define the style of words
--- Use "Input" to create different text box
--- Use "ID" name to transfer data in code
--- 使用 "WAMPSever" package to setup enviroment of "Apache; "PHP"; "MySQL" for database (关键步骤)
--- Use "Submit" input type to send data from HTML to MySQL data base
--- 有时候"Submit" 按钮可能没反应，需要更改PHP文件 地址为全称：http://localhost/LinktoDatabase.php
--- PHP 语言有大小写，比如“$_POST" 必须大小，否则不能执行,比如echarts饼图里面的“name"不能大写，否则报错
--- PHP 配置文件php.ini，要将：short_open_tag = Off #默认值改成ON，要不然不能嵌套HTML
--- HTML 不能 集成 PHP， 而PHP文件能集成HTML， 所以在首页Index.html中，直接建立一个PHP文件的跳转 window.location = xxx
--- 根据SQL select 语句得到各种需要的筛选条件结果，使用mysqli_query得到数据库数据，为 数据格式
--- 在PHP语言中：使用 json_encode 对MYSQL 获得数据进行JSON编码， (关键步骤)
--- 在HTML语言中使用Echo 函数 获得PHP 数据库查询结果，此结果可以直接被Echarts图表所使用
--- 留言板也是用数据服务器的功能，与上面的数据系统方法一致。
